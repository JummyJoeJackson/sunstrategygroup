The team page picks them up automatically — until a photo exists, the site shows
a styled initials avatar in its place, so nothing looks broken.

**Tips for best results:**

- Use portrait-orientation photos — they're displayed in a 4:5 crop.
- Aim for roughly 800 × 1000 px or larger; the image is center-cropped to fit.
- Keep file sizes reasonable (under ~300 KB each) so the page loads fast.
